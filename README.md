# Forensic Case Management System (FCMS)

## Overview
The **Forensic Case Management System (FCMS)** is a software-based solution designed to manage forensic cases efficiently and securely. The system helps investigators and authorized personnel handle case registration, evidence records, suspects, witnesses, and case progress in an organized manner.

This project was developed as an academic and practical implementation of core concepts such as system design, data management, and structured programming.

---

## Objectives
- To digitize the management of forensic cases
- To reduce manual record handling and errors
- To provide structured access to case-related data
- To improve case tracking and evidence management

---

## Key Features
- Case registration with unique Case ID
- Management of suspects and witnesses
- Evidence record handling
- User role-based access (e.g., Admin, Investigator, Lab Officer)
- Secure and organized data storage
- Simple and user-friendly interface

---

## System Modules
- **User Management**: Handles user accounts and roles
- **Case Management**: Create, update, and track forensic cases
- **Evidence Management**: Store and manage evidence details
- **Search & Retrieval**: Quickly access case information
- **Reporting**: View structured case data
